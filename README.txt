GitHub Pages – Kurzsetup

1) Neues Repo anlegen (z.B. guest-wifi)
2) Diese Dateien in den Repo-Root hochladen:
   - index.html
   - banner.png
   - favicon.ico
   - favicon.png
3) Repo → Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4) Die ausgegebene Pages-URL in der FRITZ!Box als „Webseite nach Anmeldung“ eintragen.
